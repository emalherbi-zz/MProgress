MProgress

Modal Progress Bar!

.zip .tar.gz GitHub

Basic usage

MProgress.start()

Change 'title', 'progressInc', 'progressUpdate', 'progressStriped', 'progressClass'

Example 2

MProgress.configure({
  title: 'Hello World!',
  progressInc : 1,
  progressUpdate : 0.1,
  progressStriped : true,
  progressClass : 'danger'
}).start();

window.setTimeout(function() {
  MProgress.done();
}, 60000);
